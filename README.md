# DOISR-25-26

Repository for the final project developed for the subject **Developing Open, Interoperable Semantic Resources (DOISR)** at Universidad Politécnica de Madrid.

## Project Overview

This project consists of the development of a complete ontology for the Digimon domain following Semantic Web and Linked Data principles. The ontology was designed to semantically represent Digimon entities, their stages, attributes, moves, skills, support skills and digivolution relationships.

The project also includes:

- Ontological requirements specification
- Conceptual model development using Chowlk
- OWL ontology implementation
- RDF knowledge graph generation
- SHACL validation shapes
- Ontology evaluation and FAIRness assessment
- Documentation and publication with content negotiation

## Use of Generative AI Tools

Generative AI tools such as ChatGPT Plus were used as support resources during different stages of the project, including documentation improvement, ontology conceptualization, SHACL validation support and consistency checking.

All ontology design decisions, modelling choices, implementation details and evaluation processes were reviewed and validated by the authors. The AI tool was used exclusively as a complementary aid to improve clarity, structure and consistency throughout the project.
