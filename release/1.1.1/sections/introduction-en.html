Introduction back to ToC

The Digimon Ontology was developed as part of the DOISR course project at Universidad Politécnica de Madrid. The ontology aims to provide a semantic representation of Digimon creatures and their relationships, including stages, attributes, abilities, moves and digivolution paths. The ontology was built using information extracted from publicly available Digimon datasets from Kaggle and conceptualized using Chowlk notation. The resulting ontology was implemented in OWL and validated using SHACL shapes.
